A classic HTTP & FTP server.

FEATURES
— Multi-user FTP server with username/password authentication.
— Specify a home directory for each user.
— Manage the users in a beautiful UI.
— The FTP Server listens on port 2100.
— The HTTP Server listens on port 8000.

GETTING STARTED
Install the app, and navigate to More → Apps → HTTP/FTP Server → Configure App to add your first user.

Homey Pro's IP address is listed there, so you know where to connect to.
